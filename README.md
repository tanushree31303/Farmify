# 🌾 Farmify

A farmer-friendly web application that connects **farmers** and **buyers** directly — no middlemen.

## Features

- JWT-based authentication
- Role-based access for Farmers and Buyers
- Farmers can add, update, and delete products
- Buyers can browse products, add to cart, and place orders
- Menu/category browsing

## Tech Stack

**Backend:** Java 21, Spring Boot, Spring Data JPA, MySQL, JWT  
**Frontend:** React (Vite), CSS

## Getting Started

### Backend
1. Create a MySQL database named `farmify`
2. Update `application.properties` with your DB credentials
3. Run `mvn spring-boot:run` — starts at `http://localhost:8080`

### Frontend
1. Run `npm install`
2. Run `npm run dev` — starts at `http://localhost:5173`
